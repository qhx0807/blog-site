Cannot find a valid baseurl for repo: base

出现这个问题是因为yum在安装包的过程中，虽然已经联网，但是没法解析远程包管理库对应的域名，所以只需要在网络配置中添加上DNS对应的ip地址即可。

  1. 打开 ifcfg-eth0 文件

  2. 在文件末尾追加DNS

  1. 重启网络

eth0 是网卡名称， 可以使用命令 ip addr 查看。
